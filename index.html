Artsy

A personal project inspired by Etsy, Instagram and Twitter. Front-end created using React with Redux for Hooks with Node.js with Express for the Back-end. Additional libraries include bcryptjs and jsonwebtoken for authentication, React-bootstrap for styling, and Cloudinary for photo upload.

SpaceBeez

An interactive MUD style game created in 4 days by a team of 4 Lambda School students during build week. Front-end using React and Back-end using Python with Django. Custom styling, animations and SVG icons. Additional libraries include Pusher.js which allowed for the chat functionality.

Pumpodoro

Pumpodoro was a 30 hour hackathon project created by a team of 3 Lambda School students. Pumpodoro is based on the popular Pomodoro Technique to help students to remember to get up and move during long periods of studying. The application was created using React.js & Node.js with Express for the back-end. Additional libraries include React-Heatmap-Grid and React-Router.
